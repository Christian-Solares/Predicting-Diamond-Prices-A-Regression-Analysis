# Predicting-Diamond-Prices-A-Regression-Analysis
For my project, I analyzed randomly selected 8,000 diamonds using multiple linear regression. I learned how to perform exploratory data analysis using histograms and correlation tables. However, there were issues with linearity, constant variance, and normality using residual plots. To resolve these issues, I used log-log transformation, which helped stabilize variance and normality. I also used backward elimination using AIC. I discovered that there were extremely high levels of multicollinearity in log(carat) vs. x using VIF, so I eliminated x and depth. I also used confidence intervals vs. prediction intervals. I also verified my final model, which is log(price) ~ log(carat) + cut + color + clarity. This model had an R^2 value of 0.983 with all assumptions met.
